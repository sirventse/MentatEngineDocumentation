◆ DrawRegisteredWindows()

void ME::ImGuiManager::DrawRegisteredWindows ( )
static

Renders all registered UI windows and processes pending ECS component changes.

This method iterates through all stored callbacks to draw the UI. Additionally, it handles a "pending update" queue to safely add or remove ECS components (Transform, Renderizable, Light, etc.) outside of the main systems' update loops.

Note
If a RenderizableComponent is added, it may trigger an asynchronous mesh loading task via the JobSystem.
