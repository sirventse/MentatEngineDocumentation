Getting Started

A high-performance graphics engine demonstrator.

Setup & Compilation

Follow these steps to configure the environment and build the engine.

1. Install Dependencies

The project uses Conan for dependency management. Install it with:

pip install conan

2. Generate the Project

To create the build files and the Visual Studio solution, run:

/tools/genproj.bat

3. Build Location

After running the script, the compiled solution will be available at:

build/MentatEngine.sln
